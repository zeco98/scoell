## ج) مصفوفة الصلاحيات (RBAC Matrix)

**تُنفَّذ server-side** (guards + policy checks لكل endpoint)؛ إخفاء عناصر الواجهة تجميلي فقط.

**قواعد عابرة للأدوار (server-side):**
1. عزل المستأجرين إلزامي عبر interceptor — لا استعلام بلا `tenantId` إلا لـ SUPER_ADMIN/AUDITOR.
2. الحذف النهائي للدرجات والمدفوعات: SUPER_ADMIN فقط + قيد audit حرج.
3. ترقيم السندات تسلسلي غير قابل للتلاعب (unique + معاملة).
4. كل عملية C/U/D حساسة → قيد audit بـ before/after تلقائيًا.
