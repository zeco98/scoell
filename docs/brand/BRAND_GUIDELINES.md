# دليل الهوية البصرية — منصة «منارة» (Manarah)

## 1. الاسم والقصة
- **الاسم العربي:** منارة — **اللاتيني:** Manarah
- الأسماء المرشّحة التي دُرست: **منارة** (رمز الإرشاد والمعرفة)، **مدارك** (الإدراك/التحصيل)، **رواق** (مجالس العلم). اعتُمدت «منارة» لسهولة النطق ودلالتها التعليمية القوية.
- الاسم قابل للتبديل عبر متغيّر واحد `BRAND_NAME` في `src/app/brand/Logo.tsx`.
- **الشعار (Slogan):** منصة إدارة المدارس والمعاهد.
- **نبرة الصوت:** رسمية-ودودة بالعربية الفصحى المبسّطة.

## 2. الشعار
- برمجي بالكامل (SVG) في `src/app/brand/Logo.tsx`: نسخة كاملة `LogoFull` (أيقونة + اسم) ونسخة أيقونة `LogoMark` (للـ favicon/الموبايل).
- يعمل على خلفية فاتحة (`tone="light"`) وداكنة (`tone="dark"`).

## 3. لوحة الألوان (Design Tokens في `src/styles/theme.css`)
| الدور | Token | القيمة |
|---|---|---|
| Primary (ثقة تعليمية) | `--brand` | `#0b6e63` |
| Brand Strong | `--brand-strong` | `#084f47` |
| Accent (CTAs/تمييز) | `--brand-accent` | `#f59e0b` |
| Success | `--success` | `#16a34a` |
| Warning | `--warning` | `#d97706` |
| Info | `--info` | `#0284c7` |
| Danger | `--destructive` | `#d4183d` |

- **السبب:** الأخضر الزمردي/التيل العميق يوحي بالثقة والاستقرار المؤسسي، والكهرماني يمنح تباينًا حيويًا للأزرار والتنبيهات.
- تباين النص/الخلفية مُراعى لمعيار **WCAG AA**.
- **Dark Mode:** الـ tokens معرّفة للوضعين (`:root` و`.dark`).

## 4. الخطوط
- العربية واللاتينية: **Cairo** (أساسي) مع **IBM Plex Sans Arabic** كبديل — مستوردة في `src/styles/fonts.css`.
- الأرقام: العربية الغربية (1234) افتراضيًا، مع خيار إعدادات للهندية (١٢٣٤).

## 5. نظام التصميم
- مبني على `shadcn/ui` + Tailwind v4 + Design Tokens، **RTL-native**.
- مكوّنات مشتركة: `StatCard`, `PageHeader`, `SectionCard`, `StatusPill`, `EmptyState` في `src/app/components/shared.tsx`.
